== Description ==
We have developed add-on for Yoast SEO plugin that enhances the current feature to next level.
So with the help of this add-on you can see Keywords stats , Description stats and Title stats of Yoast SEO Meta.

= Features =
1. Very Simple Clean Code plugin to list the keywords stats of yoast SEO plugin
2. Addon of yoast SEO plugin 
3. List Keywords /Description/ Title by pages/ posts .
4. Colored flags: green representing Keywords /Description/ Title used for single page/post and red representing Keywords /Description/ Title are used for multiple pages/posts.
5. Search Keywords /Description/ Title using post type and keyword.
6. Sort Keywords /Description/ Title in ascending and descending order.
7. Pagination:10 rows per page.
8. No technical skills needed.

== Installation ==
1. Upload the folder "meta-stats-yoast-seo-addon" to "/wp-content/plugins/"
2. Activate the plugin through the "Plugins" menu in WordPress
3. Go to Wp-admin-> Meta Stats Yoast -> Keyword Stats ,here you will find list of keywords stats of yoast SEO plugin
3. Go to Wp-admin-> Meta Stats Yoast -> Descriptions Stats ,here you will find list of Descriptions stats of yoast SEO plugin
3. Go to Wp-admin-> Meta Stats Yoast -> Titles Stats ,here you will find list of Titles stats of yoast SEO plugin
